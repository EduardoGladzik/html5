Tags Descontinuadas do HTML5

A partir da passagem da versão 4.1 para a 5, o HTML sofreu diversas alterações, entre elas, algumas tags, por que já não eram tão utilizadas ou por se tornarem obsoletas, foram substituidas por outras mais práticas. Aqui estão exemplos de tags descontinuadas com a atualização do HTML5.
  • <acronym>
    Essa tag definia as tags em HTML, ela foi substituida por <abbr>.
    Exemplo: <abbr> exemplo </abbr>
  • <applet>
    Definia um applet incorporado, foi substituida pela tag <object>.
    Exemplo:
  • <dir>
    Essa era usada para listar os títulos de diretório, foi substituida por <ul>.
    Exemplo:
      <ul>
      <li>
      exemplo
      </li>
      </ul>
  • <font>
    A tag <font> definia um estilo de fonte para o texto, mas foi substituida pelo uso do CSS.
    Exemplo: font-style: italic.
  • <center>
    Essa tag era responsável por centralizar os elemenos internos, entre ela. Foi substituida pelo uso do CSS.
    Exemplo: text-align: center;
  • <u>
    Já essa, era oque nos possibilitava sublinhar quaisquer palavras em uma página, ela foi substituida pela tag <ins>.
    Exemplo: <ins> exemplo </ins>
